Software Multi-Plex Custom Device Documentation


Introduction

Welcome to the documentation for the Software Multi-Plex Custom Device. This document will provide information on how to setup and use the Software Multi-Plex Custom Device.


Adding Software Multi-Plex Channels

Add Software Multi-Plex Channels via the Software Multi-Plex main page. Specify the number of channels to add in the "Number of Channels to Add" field and click the "Add Channels" button. Each channel will allow the data of one of several inputs to be dynamically directed to a single output during deployment.


Software Multi-Plex Channels

Each Multi-Plex channel consists of a single multi-plexer, containing a number of inputs, a single output, and a selector. Configuration for each of these is detailed below:

Inputs:

Select an available channel to be used as an input for the Multi-Plex and click the Map Input button to set it.

Output:

Select an available channel to be used as the output for the Multi-Plex and click the Map Output button to set it.

Selector:

During deployment, set this channel to the number corresponding to the input to be multi-plexed to the output. Values outside the number of inputs will result in no change to the Multi-Plex's current state.
