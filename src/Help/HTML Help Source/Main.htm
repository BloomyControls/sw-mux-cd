Software Multi-Plex Custom Device Documentation


Introduction

Welcome to the documentation for the Software Multi-Plex Custom Device. This document will provide information on how to setup and use the Software Multi-Plex Custom Device. Each channel of the Software Multi-Plex Custom Device routes the value of the Input channel specified by the index entered in the Selector channel to its Output channel. Selector values outside a valid range will result in no change of the Output channel.


Adding Software Multi-Plex Channels

Click on the Add Channels button to add multiplex channels. In the dialog, select the number of channels to add, the number of inputs per channel, and specify a channel name prefix and indexing scheme.


Global Selector Channel

Check the Global Selector checkbox control to have all multiplex channels select which Input to route to their Output based on a single global Selector channel. Uncheck this box to create Selector channels on a per-muliplex channel basis.
